# 🌐 Network Labs (CCNA Practice)

This repository contains hands-on networking labs designed to build practical skills in routing, switching, and network troubleshooting using Cisco technologies.

---

## 📌 About the Project

The goal of this repository is to practice and demonstrate real-world networking concepts through lab-based learning. Each lab focuses on a specific networking protocol or configuration commonly used in enterprise environments.

---

## 🛠️ Tools & Technologies

* Cisco Packet Tracer
* TCP/IP Networking
* Routing Protocols (OSPF, RIP, EIGRP, BGP)
* VLANs & Switching

---

## 📚 Labs Included

* 🔹 VLAN Configuration
* 🔹 Inter-VLAN Routing
* 🔹 OSPF (Single Area & Multi-Area)
* 🔹 RIP v2 Configuration
* 🔹 EIGRP Basics
* 🔹 DHCP Configuration
* 🔹 NAT (Static & Dynamic)
* 🔹 Basic Network Troubleshooting

---

## 🧠 Skills Gained

* Network Design & Topology Understanding
* Routing & Switching Configuration
* IP Addressing & Subnetting
* Troubleshooting using Ping & Traceroute
* Real-world lab simulation using Packet Tracer

---

## 📷 Lab Structure

Each lab includes:

* 📄 Objective
* 🌍 Network Topology
* ⚙️ Configuration Steps
* ✅ Verification / Output

---

## 🚀 How to Use

1. Download and install Cisco Packet Tracer
2. Open `.pkt` files from this repository
3. Follow the configuration steps provided in each lab
4. Test connectivity using ping/traceroute
